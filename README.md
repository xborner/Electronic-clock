# 基于 proteus 电子钟仿真

通过 keil 编译 c 文件生成 hex 文件，导入单片机，运行仿真图。

## 主要元器件：

    时钟芯片DS1302 + 温度模块18B20 +液晶显示LCD1602 + 单片机AT89C52
    
    
## 具有如下功能：

1. 自动计时，显示年、月、日、星期、时、分、秒；
2. 具备校准功能，能设置当前时间；
3. 显示当前温度。
